# TV Online Premium

Um site completo de TV online estilo Netflix/IPTV premium, totalmente responsivo, desenvolvido com HTML5, CSS3 e JavaScript puro.

## Características

- **Interface moderna**: Design escuro estilo streaming premium
- **Totalmente responsivo**: Compatível com Smart TV, Android TV, celular, tablet e PC
- **Múltiplas fontes de conteúdo**:
  - Upload de listas M3U/M3U8
  - Integração com Xtream Codes API
- **Player avançado**: Suporte para HLS (.m3u8), MP4 e TS
- **Funcionalidades completas**:
  - Busca de canais em tempo real
  - Filtro por categorias
  - Sistema de favoritos
  - Histórico de canais assistidos
  - Controles de player completos
  - Modo tela cheia
- **Persistência de dados**: Salvamento no localStorage
- **Tratamento de erros**: Canal offline, link inválido, timeout
